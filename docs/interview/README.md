# Interview Prep — Vendor Threat Watch

Public preparation material for AI agent developer interviews, built from this project. Published deliberately: every claim here is verifiable against the code, the tests, and the code-review history in this repo — reading it alongside the source is the intended experience.

| File | Use |
|---|---|
| [project-pitch.md](project-pitch.md) | 30-second and 2-minute pitches; numbers to know cold |
| [design-decisions.md](design-decisions.md) | Decision → justification pairs; the "why" behind the architecture |
| [war-stories.md](war-stories.md) | Debugging stories in tell-able form |
| [qa-prep.md](qa-prep.md) | Anticipated interview questions with answer outlines |

How to use: don't memorize scripts — rehearse the *reasoning chains* until you can reconstruct them from any starting point. Interviewers probe by pulling one thread; the value is knowing why every decision was made, not reciting that it was.
